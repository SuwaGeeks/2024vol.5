# 開発環境構築
1. `cd keyguardian-app`でディレクトリ移動
1. `docker-compose up --build`を実行
1. [localhost:3000](http://localhost:3000/)にアクセスしていつもの画面が出たら完了

# 開発環境スタート
`docker-compose up`

# 開発環境ストップ
`docker-compose stop`
